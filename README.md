# 🌐 Amar Padage Portfolio

A responsive personal portfolio website built to showcase Amar Padage's skills, education, projects, achievements, and resume.

## 🚀 Overview

This portfolio serves as a professional online presence, highlighting technical expertise, academic background, and project experience in an elegant and user-friendly interface.

## ✨ Features

* Responsive design for desktop and mobile devices
* Modern and clean UI
* About Me section
* Skills showcase
* Projects section
* Resume download option
* Contact information
* Smooth navigation

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📁 Project Structure

```text
amar-padage-portfolio/

├── images/

```

## 💻 Getting Started

1. Clone the repository

```bash
git clone https://github.com/shivraj-yadav/amar-padage-portfolio.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
